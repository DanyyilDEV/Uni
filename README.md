# Obsidian University System

A minimalist, automated workspace for university students. This repository is a pre-configured Obsidian Vault.

## Features
- **Automated Filing**: Notes are instantly moved to `01_MATERIE/{Subject}` via Templater scripts.
- **Dynamic Dashboards**: Subject Hubs (MOCs) use Dataview to aggregate lectures, concepts, and tasks.
- **Deep Study**: Dedicated templates for Concept notes using the Feynman technique with automatic backlinking.
- **Clean Architecture**: A flat folder structure designed for speed and reduced cognitive load.

## Prerequisites
The system requires the following Obsidian Community Plugins (pre-installed in the vault):
1. **Templater**: Script execution and folder management.
2. **Dataview**: Dynamic tables and automated indexing.
3. **Custom File Explorer Sorting** (Recommended): Keeps MOC files pinned at the top of folders.

## Installation
1. Download or clone this entire repository.
2. Open Obsidian and select **"Open vault from folder"**.
3. Point to the downloaded directory.
4. When prompted, click **"Trust author and enable plugins"**.
5. Go to `Settings > Templater` and ensure the Template and Script folders are correctly mapped to `_templates` and `_templater_scripts`.

# Sistema Universitario Obsidian

Uno spazio di lavoro minimale e automatizzato per studenti universitari. Questo repository è un Vault Obsidian pre-configurato.

## Funzionalità
- **Archiviazione Automatica**: Le note vengono spostate in `01_MATERIE/{Materia}` durante la creazione tramite script.
- **Dashboard Dinamiche**: Le Mappe Materia (MOC) aggregano lezioni, concetti e task tramite Dataview.
- **Studio Approfondito**: Template per Note Concetto basate sulla tecnica di Feynman con gestione dei collegamenti.
- **Architettura Pulita**: Struttura delle cartelle piatta per massimizzare velocità di scrittura e consultazione.

## Requisiti
Il sistema richiede i seguenti plugin della community (pre-installati nel vault):
1. **Templater**: Esecuzione script e gestione percorsi.
2. **Dataview**: Tabelle dinamiche e indici automatici.
3. **Custom File Explorer Sorting** (Consigliato): Mantiene i file MOC in cima alle cartelle.

## Installazione
1. Scarica o clona l'intero repository.
2. Apri Obsidian e seleziona **"Open vault from folder"**.
3. Seleziona la cartella scaricata.
4. Quando richiesto, clicca su **"Trust author and enable plugins"**.
5. Vai in `Impostazioni > Templater` e verifica che le cartelle Template e Script siano mappate correttamente su `_templates` e `_templater_scripts`.
